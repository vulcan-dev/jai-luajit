# Jai-LuaJIT
[LuaJIT](https://luajit.org/) bindings for Jai

## Building
1. Download LuaJIT from here: https://luajit.org/download.html
2. Copy `src` and `dynasm` into `./luajit`
3. Build via `jai .\generate.jai`
